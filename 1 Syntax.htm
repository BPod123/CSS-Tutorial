CSS Syntax

The syntax of CSS files appears in the following format for all of a type elements, elements of a class, or elements with an id:

            elementType { property:value; property:value;}
            elementType innerElementType { property:value;
                                            property:value;}
            elementType1, elementType2, elementType3    {property:value;} /* Multiple element types at once */
            /* This is a comment in CSS */
            .className { property:value; }
            .className elementType {property:value;}
            #idName {property:value;}
Here is an example with actual code: 
            h1 { color:blue; background-color:green;}
            h1 p {color:black; /* This sets the style for paragraphs inside of h1 headers */
                text-align:center;}
            .highlight    {background-color:yellow} /* Define the style for a class named "highlight" */

The <style> tag

You can put this code in a .css file as is, or you can put it in the head of an html file with the <style> tag as follows: 

            <head>
                <style<
                    /* CSS Code Here
                        The <style> is basically a mini .css file. */
                    h1  {color:red;}
                </style>
            </head>

Linking external .css files

Multiple web pages can use the same .css file. link your page to the file as follows: 

            <head>
                <link rel="stylesheet" href="relative/path/to/file.css">
            </head>

Inline CSS

You don't have to define the style for all elements before-hand, even though it is more scalable. You can define the style for elements in the element tag as follows:

<h4 style="property:value; property:value;"> text here  </h4>
<h4 style="property:value; property:value;"> text here  </h4>
