Text color

Set the color of the text with either, color name (if supported), rgb values, rgba values, hsl values, or hsla values as follows: 

            color:white;
            color:rgb(5, 5, 255);
            color:rgba(10, 25, 255, 100);
            color:hsl(10, 10, 10);
            color:hsla(55, 200,255, 30);
If you have a text editer like visual studio code, you may have access to a color picker that lets you choose the color and it enters the values for you.

background-color

You can set the background color of an element the same way you would the text color, but the only difference is that the property is 'background-color'. 

            <p> Normal paragraph </p>
            <p style="background-color:lightgreen"> This paragraph is lightgreen </p>

background-position and background-attachment

background-position

You can set the position of an image to the enums: 'left', 'right', 'top', 'botton', 'center', 'left center', 'right center', 'left top', 'left bottom', 'right top', 'right bottom', and any other logical combinations not listed (like 'top right');
or if you want something more specific, look at the Position tutorial.

background-attachment

Use "background-attachment:fixed" to lock the position of a background on a screen.

        <div 
        style="background-image:url('images/smileyFace.png'); background-position:right top; background-attachment:fixed; height:10vw; overflow:scroll;">

Background Shorthand

You can set the properties of a background in CSS with one line by putting the values one after another without listing the property in the format:
background: background-color background-image background-repeat background-position

        <head>
            <style>
                div {
                    background: red url('smiley.png')
                }    
            </style>
        </head>

        <body>
                <div>
                    <h1> Heading </h1>
                    <p> Paragraph </p>
                </div>
        </body>
