Shortand for borders

If you want to set the properties for all sides at once, use the format:
border: border-width border-style border-color
Only border-style is required, but if border-width or border-color are included, they must be in the correct order relative to border-style.

<p style='border:1px solid red;'> Example </p>
<p style='border thick dashed blue;'> Example </p>
<p style='border-top: thick double green; border-bottom: thin solid blue; 
    border-left: 1px solid grey;'> Example </p>
