Margin Property Values:

  • auto - Browser automatically calculates margins
  • length - px, cm, vw, mm, in
  • % - Percentage of the parent element's width
  • inherit - Margin in inherited from parent

Negative values are allowed.


Set Margins for Individual Sides 

p   {
    margin-top: 2px;
    margin-bottom: -50%;
    margin-left: 10in;
    margin-right: -5mm;
}

Margin Shorthand

You set the margins in one line in the format:
margin: top right bottom left 

p   {
    margin: 5px 20px 40px 80px;
    background-color:ligthblue;
    border:1px solid red;
}

The auto Value

Using 'auto' will put the element in the center of the parent element's margins. 


p   {
        border: 1px solid red;
        background-color:lightblue;
        width:50%;
        margin:auto;
    }


<p style='border: 1px solid red; background-color:lightblue; width:50%; margin:auto;'> Example </p>

The inherit Value

The inherit value lets you inherit the margins of the parent element.

div {
    width:50%;
    background-color:green;
    margin-left: 10px;
}
p   {
        background-color:lightblue;
        border:red;
        margin:inherit;
        width:50%;
    }

Margin Collapse

Top and bottom margins of elements on top of eachother will automatically combine to be the larger of the two so that the spacing between the elements is not the addition of the two, but left and right margins will not automatically collapse. You must set them yourself. 

.ex1    {margin: 20px 0 20px 0;
         border:1px solid red;}
.ex2    {margin: 50px 0 50px 0;
        border:1px dashed blue;
        


<p class='ex1'> Paragraph 1 </p>
<p class='ex2'> Paragraph 2 </p>
